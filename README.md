 Dataset Description

The dataset used in this project is the Ecommerce Customers Dataset. It contains information about customers of an online e-commerce company, including their behavior and spending patterns.

🔹 Features included:
Avg. Session Length: Average time spent during in-store style advice sessions.
Time on App: Time spent on the mobile application.
Time on Website: Time spent browsing the website.
Length of Membership: Duration (in years) that the customer has been a member.

 Target Variable:
Yearly Amount Spent: The total amount of money a customer spends in a year.

 Dataset Purpose

This dataset is used to analyze customer behavior and identify the key factors that influence how much customers spend annually. It is also used to build a predictive model for estimating customer spending.

 Data Preprocessing
Checked for missing values and cleaned the dataset.
Selected relevant features for analysis.
Split the data into training and testing sets.
 Model Usage

A regression model was applied to predict the yearly amount spent by customers based on their behavior.

 Goal of the Project

The main goal is to understand customer behavior and provide business recommendations to increase revenue, especially by improving high-impact areas like the mobile app.
